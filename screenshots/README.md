# Screenshots cần chụp

Chụp từ app trong `example/` (chạy `flutter run`), lưu đúng tên file dưới đây vào thư mục này:

| File | Nội dung |
|---|---|
| `date_picker.png` | Date Picker — chế độ portrait |
| `time_picker.png` | Time Picker — scroll wheel 12h/24h |
| `date_time_picker.png` | Date-Time Picker — tab Date / Time |
| `range_picker.png` | Date Range Picker — dải band highlight |
| `year_grid.png` | Year Grid overlay (bấm vào header) |

Gợi ý:
- Chụp trên thiết bị/emulator có tỉ lệ khung hình phổ biến (ví dụ khung điện thoại 1080x2340 hoặc crop lại vùng dialog cho gọn).
- Nén ảnh trước khi commit (mỗi ảnh nên < 500KB) để repo không quá nặng — có thể dùng `pngquant` hoặc export JPG chất lượng ~85%.
- Sau khi có đủ ảnh, xóa file README.md này (nó không cần thiết khi publish).
